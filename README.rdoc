== Install and use

=== 1. Install the Authlogic Easy Token plugin

  $ script/plugin install git://github.com/ababich/authlogic_easy_token.git
  
=== 2. Now it works

  Your perishable_tokens become easy - 10-chars strings - manual input and even remember it is not so hard

=== Notes

  You should realize security risks using Easy Tokens.
  Recommended usage: activation via SMS or smth like this that may require manual input.
